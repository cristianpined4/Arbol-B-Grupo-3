# Arbol-B-Grupo-3
Tarea #2 de Estructura de datos UES 2022.


    REQUERIMIENTOS:
    
    1 - Insertar claves en un árbol B.
    2 - Buscar claves en un árbol B.

    5 - Mostrar índices del árbol.
